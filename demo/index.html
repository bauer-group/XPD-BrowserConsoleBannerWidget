Console Security Banner

Ein schlanker, zweistufiger Banner, der Endnutzer direkt in der Browser-Konsole vor Self-XSS-Betrug warnt — und nur dann erscheint, wenn die DevTools tatsächlich geöffnet werden.

DevTools schließen-Zustand wird beobachtet … Öffnen Sie die Konsole mit F12 bzw. ⌘+⌥+I — der Banner erscheint dort.

Warum zweistufig?

Stage 1 · Loader

~1,2 KB gzip, eager. Null Netzwerk-Requests, solange die Konsole geschlossen ist.

Stage 2 · Banner

Lazy. Wird erst bei erkannten DevTools geladen — inkl. 22 Sprachen, einmal pro Tab.

Kein Tracking

Keine Cookies, keine Telemetrie, kein Render-Blocking. Reiner Schutzhinweis.

Diese Demo lädt Loader & Banner relativ aus ../dist/v1/. Vorher einmal npm run build ausführen, dann das Repo-Root mit einem Static-Server ausliefern und /demo/ öffnen.
